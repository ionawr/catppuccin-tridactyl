## Usage

To apply a theme to Tridactyl, use the `:colourscheme` command. Replace `<flavor>` with your desired flavour.
For example, to set the Mocha theme:

```
:colourscheme --url https://raw.githubusercontent.com/ionawr/catppuccin-tridactyl/main/themes/catppuccin-mocha.css catppuccin-mocha
```
